## Target

- 💡 Vue or React와 같은 **개발 툴을 사용하는 이유**가 궁금한 개발자

- 💡 SPA를 어디서 들어봤지만, **명확한 의미**를 모르는 프론트 개발자

- 💡 프론트엔드 **프레임워크 원리**가 궁금한 백엔드 개발자
