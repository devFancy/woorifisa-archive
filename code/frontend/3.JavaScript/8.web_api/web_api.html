BOM(Browser Object Model)

브라우저 객체 모델, 브라우저 자체에 대한 몇 가지 조작이 가능, 표준은 따로 없고 각 브라우저 벤더별로 서로 다른 구현 스펙을 가짐.

DOM(Document Object Model)
