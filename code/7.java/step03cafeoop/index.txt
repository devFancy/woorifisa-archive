카페 예제
요구사항


손님(Customer)
손님은 음료를 주문할 수 있음
손님은 음료 주문 시 금액을 결제, 지불해야함
손님이 가진 돈 보다 높은 금액의 음료는 주문 불가능
손님이 음료를 마시면 음료는 줄어듦

캐셔(Casher)
캐셔는 음료를 주문받을 수 있음
손님으로부터 금액을 전달받고, 돈통에 넣어야함
바리스타에게 전달받은 음료를 고객에게 전달함

바리스타(Baristar)
바리스타는 캐셔로부터 음료 주문을 전달받음
음료 주문을 보고 음료를 제조함
제조가 완료되면 캐셔에게 음료를 전달함

사장(Owner)
돈통을 조회할 수 있음
하루 총 매출 금액 조회 기능

카페(Cafe)
메뉴(Menu) - 카페 클래스의 구성요소
메뉴 아이템(MenuItem) - 메뉴 클래스의 구성요소
음료는 아메리카노, 스프라이트, 사과유자차, 페퍼민트가 있음

주문 정보(OrderInfo)
주문 상품명
주문 고객 정보
주문 시간

영수증(Recipe)
거래 날짜
주문 내용
- 주문 상품

음료(Drink)
바리스타가 이용할 수 있는 클래스
사이즈(size),
음료 종류(ex. 레시피 클래스 상속받은 아메리카노, 스프라이트 등)

레시피(Recipe)
재료 리스트
제조법 설명

주문 상품(Product)
캐셔, 고객이 이용할 수 있는 클래스
필드 구성
주문 음료(Drink Product)
영수증(Recipe)
쟁반 제공 여부
